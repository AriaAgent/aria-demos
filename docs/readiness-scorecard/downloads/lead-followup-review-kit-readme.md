## 这个产品包在卖什么

面向有销售/咨询线索的小团队，做一个轻量体检：

- 画出线索从进入到跟进的流程；
- 找出最容易漏单/拖慢/缺信息的 3 个位置；
- 标出 AI 可以做什么、不能做什么；
- 给一个一周内可人工复核的小试点建议。
